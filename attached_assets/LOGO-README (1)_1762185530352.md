# FundRazor Logo Files

## Logo Files Included

### 1. SVG Files (Ready to Download)
- **FundRazor-Logo.svg** - Dark version (ice blue on transparent, for navy backgrounds)
- **FundRazor-Logo-Light.svg** - Light version (dark blue on transparent, for white backgrounds)

### 2. React Component
- **FundRazorLogo.tsx** - Production-ready React component with variant support

## Usage

### Using SVG Files
Simply download and use the SVG files in your projects:
```html
<img src="FundRazor-Logo.svg" alt="FundRazor" />
```

### Using React Component
```tsx
import { FundRazorLogo } from './FundRazorLogo';

// Default (dark variant)
<FundRazorLogo />

// Light variant (for white backgrounds)
<FundRazorLogo variant="light" />

// Custom size
<FundRazorLogo width={400} height={114} />

// With Tailwind classes
<FundRazorLogo className="w-64" />
```

## Brand Specifications

### Colors

**Dark Background Version (Primary)**
- Primary Text (Fund): `#FFFFFF` - White
- Accent (Razor + Slash): `#7DD3FC` - Ice Blue
- Recommended Background: `#0A1628` to `#1A3A5C` - Regal Navy Blue

**Light Background Version**
- Primary Text (Fund): `#0F172A` - Dark Slate
- Accent (Razor + Slash): `#0284C7` - Sky Blue
- Recommended Background: `#FFFFFF` - White

### Typography
- Font Family: `system-ui, -apple-system, sans-serif`
- Font Weight: `700` (Bold)
- Letter Spacing: `-0.02em` (Tight)

### Design Elements
- Slash Direction: Upward (↗) - Bottom-left to top-right
- Slash Style: Double line with opacity variation for depth
- Concept: "Precision Cut" - representing sharp, cutting-edge AI intelligence

## File Specifications

### SVG Dimensions
- Width: 280px
- Height: 80px
- Aspect Ratio: 3.5:1

### Scaling
The logo scales proportionally. Maintain aspect ratio when resizing.

## Brand Guidelines

### ✓ Do
- Use on navy blue backgrounds for primary branding
- Use light version on white or light backgrounds
- Maintain clear space around the logo (minimum 20px)
- Scale proportionally
- Keep colors consistent with brand specifications

### ✗ Don't
- Change the slash direction or angle
- Alter the color scheme
- Distort or stretch the logo
- Add effects, shadows, or outlines
- Place on busy or low-contrast backgrounds
- Separate "Fund" from "Razor"

## Converting to Other Formats

### To PNG (using Figma, Illustrator, or online converters)
1. Open the SVG file
2. Export as PNG at your desired resolution (recommend 2x or 3x for retina displays)
3. Suggested sizes:
   - Small: 280×80 (1x)
   - Medium: 560×160 (2x)
   - Large: 840×240 (3x)

### To Favicon
1. Create a square version (crop to 80×80 showing just "FR" or use a simplified icon)
2. Export as PNG at 512×512
3. Use a favicon generator to create .ico and various sizes
